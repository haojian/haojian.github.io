##database
===============


#### Common SQL commands

```
// bash terminal
mysql -u USERNAME -h HOSTNAME -p
mysql -u root -p		# login a local database server
mysqldump ghost | mysql ghost-main	# clone database; may need to add the root access and password setting
```

```
show databases;
```
